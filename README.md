# qc-skills

青城自建 AI Skill 合集，用于知识树协作系统。12 个 Skill，覆盖知识管理、内容生产、系统维护三大领域。

## 技能列表

### 知识管理（4）
| 名称 | 版本 | 用途 |
|------|------|------|
| 知识调用 | v2.0 | 回答前自检→扫描 MOC→三路径召回，强制记录调用 |
| 知识编译 | v3.4 | 三阶段流水线：深度整理→卡片化→全量互联 |
| 知识库体检 | v2.0 | 13 步全库健康检查（含网络检测、生命周期、张力审查） |
| 教学对话 | — | AI 主导交互式学习，基于知识库卡片 |

### 运转与沉淀（3）
| 名称 | 版本 | 用途 |
|------|------|------|
| 对话存档 | v2.1 | 两层格式存档，6 点为界，整理修订幂等标记 |
| 日报 | v1.0 | 每日跨日快照：知识库状态、本周轨迹、跨日趋势 |
| 周复盘 | v1.0 | 本周 MOC 生长分布、调用记录定性补充、被低估卡片 |

### 系统维护（3）
| 名称 | 版本 | 用途 |
|------|------|------|
| 协作框架审核 | v2.1 | 10 步审核宪法/Skill/MOC/软链接/Token 审计 |
| 工具管理 | v1.1 | 清点/安装/体检/评估四合一，中立文件夹归位 |
| Skill自修正 | v2.1 | 用户纠正→根因分析→提议修正→验证闭环 |

### 内容与数据（2）
| 名称 | 版本 | 用途 |
|------|------|------|
| 写作方法自进化 | v1.0 | 对比 AI 原稿↔终稿 diff，跨对找共性修改规律 |
| WeFlow记录处理 | v1.0 | 微信聊天记录双层清洗：结构解析 + 大模型语义过滤 |

## 变更记录

- 2026-06-14：全量更新到最新版。新增 Skill自修正（替代 Skill自动生长）。移除知识网络质量检测（合并入知识库体检）、系统精简（合并入协作框架审核）。12 个 Skill。

## 安装

```bash
npx skills add qc-sidios/qc-skills
```
