# qc-skills

青城自建 AI Skill 合集，用于知识树协作系统。

## 技能列表

| 名称 | 用途 |
|------|------|
| 知识调用 | 回答前扫描 MOC，定位相关知识卡片 |
| 知识编译 | 外部知识编译为原子卡片，三阶段流水线 |
| 教学对话 | AI 主导交互式学习，基于知识库 |
| 知识库体检 | 审计全库元数据、链接、生命周期 |
| 知识网络质量检测 | 6 维定量 + TypeLink 定性分析 |
| 写作方法自进化 | 对比原稿终稿，自动学习写作偏好 |
| 对话存档 | 两层格式存档对话，整理修订 |
| 日报 | 每日跨日状态快照与演化轨迹 |
| 周复盘 | 每周 MOC 生长 + 调用补充 + 建议 |
| 协作框架审核 | 审计宪法/Skill 版本号与格式 |
| 系统精简 | Token 审计与冗余扫描 |
| 工具管理 | 清点/安装/体检/评估四合一 |
| Skill自动生长 | 重复 3 次，自动提议固化 Skill |
| WeFlow记录处理 | 双层高精清洗微信聊天记录 |

## 安装

```bash
npx skills add qc-sidios/qc-skills
```
