# WeFlow 记录处理 Skill (双层高精过滤管道)

## 这是什么

这是一个将 WeFlow 导出的纯文本（`.txt`）微信聊天记录文件，进行**结构重构、微信 ID 翻译与大模型语义去噪**的原地一键自动化清洗工具。

### 核心设计
1. **原地就地修改（In-place Writeback）**：清洗、过滤后的内容将**直接覆盖写入原来的 `.txt` 文件**，不会产生多余的 Markdown 文件，杜绝混乱。
2. **安全双保险**：在处理任何原始文件前，系统会**自动复制生成一份 `.bak` 备份文件**，确保绝对的数据安全。
3. **两层过滤架构 (SOP 对齐)**：
   * **Layer 1 (结构解析与翻译)**：通过正则剔除撤回、加入群聊等系统硬噪音；将 `wxid_xxx` 翻译为真实昵称；进行天之内的消息核心去重。
   * **Layer 2 (预清洗 + 大模型语义过滤)**：在本地程序化拦截刷屏打卡和结构化破冰模板；通过 DeepSeek-v4-flash (关闭思考模式) 对其余聊天消息进行高精语义判别，根据九大保留标准（包含教练点评、技术Q&A、深度话题、常青链接白名单等）和五大删除标准过滤 95% 以上闲聊废话。

## 何时触发

当青城发出以下指令或提及相关词时：
- “帮我把刚导出的 WeFlow 记录清洗一下”
- “过滤微信群聊记录”
- “处理新微信记录并脱水”
- “对素材箱的聊天记录进行去噪”

---

## 用法说明

### 核心命令

使用 Skill 自带的管道启动器 `scripts/run_pipeline.sh` 进行一键串联运行：

```bash
bash scripts/run_pipeline.sh "<WeFlow 聊天记录 .txt 文件路径或目录路径>"
```

#### 运行示例

##### 1. 清洗单个文件
```bash
bash scripts/run_pipeline.sh "/Users/chenyilin/Documents/青城的知识树/素材箱/_待处理/微信记录/群聊_DeepSeek入门2群-3月航海_全部时间.txt"
```

##### 2. 批量清洗目录下所有符合条件的群聊/私聊 `.txt` 记录 (推荐)
```bash
bash scripts/run_pipeline.sh "/Users/chenyilin/Documents/青城的知识树/素材箱/_待处理/微信记录"
```

运行完成后，脚本会输出直观的降噪百分比汇总报告，并将原始记录原地覆盖，备份存储为 `.bak` 文件。
